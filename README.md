## Проект в рамках соревнований

В составе команды **RoboFox** принимала участие в разработке решения для задачи автономного передвижения.

**Моя роль:**

* разработка модели для распознавания объектов
* фокус на распознавании дорожных знаков

**Что было сделано:**

* сбор и подготовка датасета (разделение на train/test)
* обучение модели на изображениях
* проведение экспериментов и обучение модели в Google Colab

**Используемые технологии:**
* Python
* компьютерное зрение
* машинное обучение
* Google Colab
