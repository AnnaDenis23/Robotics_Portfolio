* СГУГиТ, Новосибирск
* Статья опубликована в сборнике материалов конференции

📄 [Открыть сборник (страница 64)](https://sgugit.ru/upload/science-and-innovations/konferentsii-i-nauchnye-meropriyatiya/inzhenernaya-grafika-i-trekhmernoe-modelirovanie/Sbornik_2026.pdf)

**О статье:**
Работа посвящена решению задачи в области инженерной графики и 3D-моделирования.
В рамках исследования были рассмотрены методы построения и анализа моделей, а также практическое применение инструментов инженерной графики для решения научной задачи.

Особое внимание уделено:

* построению корректной геометрической модели
* анализу точности и применимости решения
* использованию современных инструментов моделирования

## Проект в рамках соревнований

В составе команды **RoboFox** принимала участие в разработке решения для задачи автономного передвижения.

**Моя роль:**

* разработка модели для распознавания объектов
* фокус на распознавании дорожных знаков

**Что было сделано:**

* сбор и подготовка датасета (разделение на train/test)
* обучение модели на изображениях
* проведение экспериментов и обучение модели в Google Colab

**Используемые технологии:**
* Python
* компьютерное зрение
* машинное обучение
* Google Colab
